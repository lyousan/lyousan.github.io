Selenium 介绍

Selenium[ /səˈliːniəm/ ] 是一个用于Web应用程序测试的工具。Selenium测试直接运行在浏览器中，就像真正的用户在操作一样。支持的浏览器包括IE（7, 8, 9, 10, 11），[Mozilla Firefox](https://baike.baidu.com/item/Mozilla Firefox/3504923)，Safari，Google Chrome，Opera，Edge等。这个工具的主要功能包括：测试与浏览器的兼容性——测试你的应用程序看是否能够很好得工作在不同浏览器和操作系统之上。测试系统功能——创建回归测试检验软件功能和用户需求。支持自动录制动作和自动生成[ .Net](https://baike.baidu.com/item/ .Net/156737)、Java、Perl等不同语言的测试脚本。

简单的来说，它可以模拟用户对浏览器的操作，包括点击、输入等等的交互操作都可以通过编程来实现，并支持各种不同的语言，

我们可以借此实现自动化的测试，当然也能用来抓取网页中的数据

这里需要注意的是：

  • 设置系统参数一定要在创建驱动之前，否则会报错

  • 在进行模拟操作时，建议在程序中加上一定的休眠时间，否则太快、太多的操作可能会触发一些网站的防御机制

  • 如果不希望每次创建驱动时都手动设置系统参数，可以将驱动文件（chromedrive.exe）移动到jdk环境变量的目录中，也就是bin目录中

总结

如果你对html或js有一些了解的话，那么selenium是很容易上手的，你会发现其核心逻辑就是操作dom，获取元素，然后根据业务需求进一步获取元素的内容和属性，难点就在于如何获取节点，selenium也提供多种获取元素的选择器，灵活使用能让你每天都准点下班😉，上班划水利器。
